# Local Monitoring Stack

A lightweight local monitoring environment built with:

* Vagrant
* Docker Compose
* Prometheus
* Grafana
* Ansible

This project was created as a DevOps test assignment focused on deploying and configuring a local monitoring stack.

---

## Features

* Local VM provisioning using Vagrant
* Docker Compose based service orchestration
* Prometheus monitoring setup
* Grafana dashboard integration
* Infrastructure automation using Ansible
* Lightweight local development environment

---

## Stack

* Vagrant
* Docker
* Docker Compose
* Prometheus
* Grafana
* Ansible

---

## Quick Start

### 1. Clone repository

```bash
git clone <repository_url>
```

### 2. Enter project directory

```bash
cd Test_for_HF
```

### 3. Start environment

```bash
vagrant up
```

### 4. Open Grafana

```text
http://localhost:3000
```

Default credentials:

```text
admin / admin
```
